## 使用

```shell
  使用方式
    $ 指令 [输入] [选项]

  选项
    --version, -v                              查看版本号
    --help, -h                                 查看帮助

  命令
    set                                        设置 指令 全局设置
    reset                                      清空 指令 全局设置
    defaults                                   查看 指令 全局设置

  示例
    $ 指令                                      查看当前帮助说明
```

### NPM

```shell
npm i 包名 --global
```

### PNPM

```shell
pnpm add 包名 --global
```

### Yarn

```shell
yarn global add 包名
```

上述都安装、配置好后，就可以顺利使用 "包名" 了。😃
